The maximum value for the plot geometry represents the vector count or percent represented by the outer circle of the plot. By changing this value, the data within the plot are rescaled. Decreasing the value will show more detail, and increasing the value will show less detail.

To manually change maximum value,

1. Open the inspector panel (click on the toolbar icon).

2. Click once in the plotting region of a document window.

3. Select "Grid System" in the uppermost popup button.

4. Enter value in the Maximum Value box or use the stepper to increment through values. (Note: if percent units are used, this number should be between 0 and 100).

To automatically calculate maximum value,

1. Proceed with steps 1-3 above.

2. Click on the Auto Calculate Max Value button.
Notes: The grid system maintains different maximum values for both unit types, count and percent, since these values are not directly convertable if multiple layers exist. Thus, changing the units will change the maximum value.
