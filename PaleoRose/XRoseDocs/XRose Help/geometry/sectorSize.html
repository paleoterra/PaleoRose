A sector is the angular "bin" defined by a start and end angle. Any vector that falls between a sector's angle are collected in the bin. The sectors then appear in the plot as a line, petal, or series of counting dots.

To change sector size,

1. Open the inspector panel (click on the toolbar icon).

2. Click once in the plotting region of a document window.

3. Select "Sector System" in the uppermost popup button.

4. Sector size can be set by either angle or by count. Enter the desired value in the corresponding text box, or use the stepper to step through valid angles.
Notes: It is important to recognize that some angles will generate irregular sector increments. However, XRose does not allow any of these invalid angles. If you enter an invalid angle, it will automatically choose a valid angle close to the one you selected.
