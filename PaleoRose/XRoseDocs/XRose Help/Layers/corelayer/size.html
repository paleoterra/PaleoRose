If the core layer is not linked to the hollow core, then the core layer's size can be set manually.

To set core size,

1. Open the inspector panel (click on the toolbar icon).

2. Select the core layer in the layer table.

3. Select "Size" in the uppermost popup button.

4. Set size by either using the relative size slider or set a number manually in the text box.
