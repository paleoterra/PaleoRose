If the core layer is linked to the hollow core, the size of the core is determined by the size of the hollow core (see the plot geometry inspector) and will have a special fill pattern.

To link to the hollow core,

1. Open the inspector panel (click on the toolbar icon).

2. Select the core layer in the layer table.

3. Select "Size" in the uppermost popup button.

4. Click on the "Link To Hollow Core Switch".
Note: If the hollow core size is set to "0", then the core layer will seem to disappear. Reset the hollow core size to change the size.
