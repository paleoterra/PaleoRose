The default colors of the core are black lines with while fill, unless linked to the hollow core size which requires a patterned fill.

To set colors,

1. Open the inspector panel (click on the toolbar icon).

2. Select the core layer in the layer table.

3. Select "Appearence" in the uppermost popup button.

4. Set the color or the line or fill using the appropriate color well.

To set line and fill colors to the same color,

1. Follow steps 1-3 above.

2. Click on the line color and then shift click on the fill color.

3. Use the color panel to set the colors.
