Core Layer

The number of circles in the plot can be either fixed or variable. When the circle count is fixed, if the count or percent of the outer ring is changed by the user, the plot still has the same number of circles. However, if the count is not fixed, then the circle count becomes depended on the count or percent of the outside ring. Variable circle counts are set by choosing a frequency, as either a count or percent. Thus, increasing the count or percent of the outer ring will add circles to the plot.

Working with fixed count rings,

1. Open the inspector panel (click on the toolbar icon).

2. Click on the grid layer in the layers table.

3. Select "Circles" in the uppermost popup button.

4. If not already selected, turn on the "Fixed Count" switch

5. Set the desired number of circles with "Variable Ring Increments" text box or stepper.

Working with variable rings,

1. See steps 1-3.

2. Ensure the "Fixed Count" switch is turned off.

3. Set the desired frequency in the "Value Increments" fields.

If your grid system is in percents, fill in a percent value or use the stepper. If your grid system is in counts, use the count stepper and text box.
