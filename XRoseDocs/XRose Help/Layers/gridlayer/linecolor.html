Core Layer

Grid lines only support line color and cannot be filled. Changing the line color will change the color of the radial and circle lines, and any displayed labels.

To change line color,

1. Open the inspector panel (click on the toolbar icon).

2. Click on the grid layer in the layers table.

3. Select "Appearence" in the uppermost popup button.

4. Click on the color well and select the color.
