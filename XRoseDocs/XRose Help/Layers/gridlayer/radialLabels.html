Showing and Customizing Labels

Core Layer

Labeling tick marks in XRose is somewhat complex. You can control font and font size, the numbering system, alignment, and compass point types.

Numbering:

0-360: This method displays the radial angle from 0 (North) clockwise to just before 360.

Quadrant: From North, values are counted up from 0 to 90 degrees in both clockwise and counterclockwise directions. Values are counted in the same manner from South.

Points Only: Shows only the compass points: 0, 90, 180, and 270

Alignment:

Horizontal: All labels are parallel with the East-West line.

Angle: Each label is parallel with their respective radial lines.

Points:

Letters: Shows the major compass points as N,S, E, and W.

Numbers: Shows all major compass points at numbers.

To change labels,

1. Open the inspector panel (click on the toolbar icon).

2. Click on the grid layer in the layers table.

3. Select "Radial Ticks and Labels" in the uppermost popup button.

4. Make the desired selections in font, numbering, alignment, and compass point labels.
