Dot and dot deviation plots will allow the user to define circle size.

To change circle size,

1. Open the inspector panel (click on the toolbar icon).

2. Choose the data layer in the layers table.

3. Select "Appearence" in the uppermost popup button.

4. User the circle size slider to set an appropriate circle size.
