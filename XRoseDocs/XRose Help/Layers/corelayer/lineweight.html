Set Line Weight

The line weight of the circle is independent of all other layers and can be set by the user.

To line weight,

1. Open the inspector panel (click on the toolbar icon).

2. Select the core layer in the layer table.

3. Select "Appearence" in the uppermost popup button.

4. Use the line weight slider or text box to set the line weight.
