The core layer is a circle with a solid fill. It has two rolls. First, it can be used to overlay the near zero portion of the plot to clean up the appearence of the plot. The second role is connected to the hollow core (set in the Plot Geometry inspector). In this case, the core has a special fill pattern and the size of the plot is directly connected to the size of the hollow core.
