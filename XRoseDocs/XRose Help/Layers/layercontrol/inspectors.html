Each layer type has it's own custom inspector. An inspector is a window in which you can examine and evaluate properties of a system. In this case, each layer inspector contains the settings for each individual layer.

To see a layer inspector,

1. Make sure the inspector window is visible. Click on the inspector toolbar button if the window is not visible.

2. Select the desired layer in the layer table.
Note: ALPHA VERSION - I haven't found a satisfactory method for switching from the geometry inspector to the layer inspector. You will have to deselect all layers and reselect the desired layer.
