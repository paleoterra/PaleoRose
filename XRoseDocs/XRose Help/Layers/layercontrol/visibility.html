Some layers have control of partial visibility within their property inspectors. However, all layers may be turned off from the layer table. Layers that are turned off remain in memory and can be turned back on, as opposed to deleting the layer that destroys the data.

To show or hide a layer,

Click on the switch next to the layer name. If the switch is on, the layer should be visible.

Note: this feature may appear to fail if you have turned off visibility for all the subfeatures in a particular layer. See documentation on data layers..
